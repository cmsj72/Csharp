1. 표준 출력

   - (System.)Console.Write(), Console.WriteLine()→WriteLine은 개행을 함

   

2. 출력 형식

   - Console.Write(변수 또는 데이터);

   - Console.Write("{0} {1}", 변수1, 변수2);

   - 출력 형식 문자→MSDN 참고

     | 형식문자 |              내용              |
     | :------: | :----------------------------: |
     |    C     |    통화($,원화 기호등) 표시    |
     |    D     |          10진수 정수           |
     |    E     |         지수형태 출력          |
     |    F     |        부동소수점 출력         |
     |    G     |           기본 출력            |
     |    N     | 콤마 출력(ex: 1,000 같은 형태) |
     |    P     |         % 단위로 출력          |
     |    X     |          16진수 출력           |
